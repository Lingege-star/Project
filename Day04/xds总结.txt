git 初始化命令  只是第一次初始化下
git init

将工作空间里面所有的文件和文件夹都提交到暂存区    
git add 

将暂存区里面的文件提交到本地仓库
git commit -m "注释"

查看状态的命令                                            
git status

查看每次提交到本地仓库的版本号   （版本号是随机生成的一串字符串）
git reflog

回退到上一个版本 
git reset --hard HEAD^....  
 
回退到某个具体的版本号      
git reset --hard  版本号
     
设置关联远程仓库
生成ssh密钥
ssh-keygen -t rsa -C "1119633919@qq.com"

git关联远程仓库命令
git remote add origin 远程仓库位置

git抽取远程仓库所有文件
git clone  远程仓库位置

把本地仓库的项目提交到远程仓库
git push -u origin 分支名字

查看但是分支数量
 git branch

创建+切换分支
git checkout -b 新建分支名字

切换分支
git checkout 分支名

查看分支 
git branch

创建分支
git branch name

删除分支
git branch Cd name

如果上面拉取代码的方式不行，只需要创建一个空目录 然后通过   
git clone https://github.com/Thierry1/Android-H2003B.git


切换到主分支然后合并子分支
合并某分支到当前分支
git merge 子分支
