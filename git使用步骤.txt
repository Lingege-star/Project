第一天到公司:
1.git 初始化命令  只是第一次初始化下
	git init
2.克隆
	git clone 仓库地址 例如：git@github.com:15372992326m/sk.git
3.重新打开 查看状态
4.切换分支
	 git checkout -b dev
5.修改文件
6.状态 
	git status
7.添加 
	git add .
8.注释 
	git commit -m "注释"  
9.状态 
	git status
10.切主分支 
	git checkout 分支名
11.合并 
	git merge dev
12.关联仓库 
	git remote add origin 仓库地址
13.把本地仓库的项目提交到远程仓库
	git push -u origin master（可变）	(用户名 密码)
